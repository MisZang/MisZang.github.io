请注意：由于浏览器安全限制，您选择的二维码图片无法直接被网页文件修改和重命名。

如果您将此项目托管在 GitHub 等平台，您必须在**本地**完成以下操作：

  1. 将项目仓库克隆或下载到您的电脑。
  2. 选择新的二维码图片。
  3. 手动将新图片重命名为 **WeChatQC.png**。
  4. 将重命名后的图片替换到项目文件夹的 **图片/二维码/** 路径下。
  5. 将修改提交并推送回您的 GitHub 仓库。
